# Legacy PRD: Universal Interface

## Status

This PRD is paused and retained for historical reference. It was written for the earlier Chorus vision as a broad safety layer for decentralized agent ecosystems.

## Why It Is Not The Current Product Definition

The active Chorus MVP is narrower and more concrete:

- secure delegated actions
- per-agent capability grants
- approval-aware execution
- audit and quarantine visibility

That shift is captured in the active documents:

- [Product Overview](product_overview.md)
- [Auth Control Plane Requirements](specs/auth-control-plane/requirements.md)
- [Auth Control Plane Design](specs/auth-control-plane/design.md)

## Future Reuse

If Chorus later expands beyond the current Gmail and GitHub control-plane demo, some universal-interface ideas may become relevant again. Until then, this PRD should be treated as an archived exploration, not an implementation target.
